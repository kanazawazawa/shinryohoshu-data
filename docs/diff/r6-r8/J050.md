# J050 改定差分 (R6 → R8)

- **R6 名称**: 気管内洗浄（１日につき）  (点数: `425`)
- **R8 名称**: 気管内洗浄（１日につき）  (点数: `510`)

## 構造差分

- 所定点数: `425` → `510`

## R6 原文

```
Ｊ０５０気管内洗浄（１日につき）425点

- ２気管内洗浄と同時に行う喀痰吸引又は酸素吸入は、所定点数に含まれるものとする。
```
