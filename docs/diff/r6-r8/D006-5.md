# D006-5 改定差分 (R6 → R8)

- **R6 名称**: 染色体検査（全ての費用を含む。）  (点数: `2477`)
- **R8 名称**: 染色体検査（全ての費用を含む。）  (点数: `2477`)

## 構造差分

- 注1 加算点数変更: 分染法加算 397 → 386

## R6 原文

```
Ｄ００６－５染色体検査（全ての費用を含む。）１ＦＩＳＨ法を用いた場合2,477点
- ２流産検体を用いた絨毛染色体検査を行った場合4,603点
- ３その他の場合2,477点
- 注１分染法を行った場合は、分染法加算として、397点を所定点数に加算する。
- ２２については、別に厚生労働大臣が定める施設基準に適合しているものとして地方厚生局長等に届け出た保険医療機関において行う場合に限り算定する。
```

## R8 原文

- ２流産検体を用いた絨毛染色体検査を行った場合4,465点
- ３その他の場合2,444点
- 注１分染法を行った場合は、分染法加算として、386点を所定点数に加算する。
- ２２については、別に厚生労働大臣が定める施設基準に適合しているものとして地方厚生局長等に届け出た保険医療機関において行う場合に限り算定する。
