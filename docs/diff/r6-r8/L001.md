# L001 改定差分 (R6 → R8)

- **R6 名称**: 筋肉注射による全身麻酔、注腸による麻酔  (点数: `120`)
- **R8 名称**: 吸入麻酔又は静脈麻酔による鎮静  (点数: `None`)

## 構造差分

- 名称: 「筋肉注射による全身麻酔、注腸による麻酔」→「吸入麻酔又は静脈麻酔による鎮静」
- 所定点数: `120` → `None`

## R6 原文

```
Ｌ００１筋肉注射による全身麻酔、注腸による麻酔120点
```

## R8 原文

```
Ｌ００１吸入麻酔又は静脈麻酔による鎮静
- １10分未満のもの120点
- ２10分以上20分未満のもの310点
```
