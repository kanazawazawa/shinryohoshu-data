# D241 改定差分 (R6 → R8)

- **R6 名称**: 神経・筋検査判断料  (点数: `180`)
- **R8 名称**: 神経・筋検査判断料  (点数: `None`)

## 構造差分

- 所定点数: `180` → `None`

## R6 原文

```
Ｄ２４１神経・筋検査判断料180点
- 注神経・筋検査等の種類又は回数にかかわらず月１回に限り算定するものとする。
```

## R8 原文

```
Ｄ２４１神経・筋検査判断料
- １筋電図検査の場合180点
- ２その他の場合180点
- 注神経・筋検査等の種類又は回数にかかわらず月１回に限り算定するものとする。
ただし、いずれかの神経・筋検査判断料を算定した場合には、同一月において他の神経・筋検査判断料は算定しない。
```
